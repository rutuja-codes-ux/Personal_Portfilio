## ✦ Overview

This is a professional, high-end personal portfolio website designed to showcase my skills, experience, and projects. It is built with a focus on modern web design aesthetics, specifically aiming for a **"Dark Editorial Luxury"** theme. The site is a single-page application with smooth scrolling and dynamic visual effects that provide a premium user experience.

## ✨ UI/UX & Design Philosophy

The design prioritizes visual excellence and a dynamic, interactive experience. Key UI/UX features include:

- **Dark Editorial Luxury Aesthetic**: A sophisticated dark mode (with a light mode toggle) that uses a curated color palette (deep blacks, muted greys, and vibrant neon accents like `#00FFB2` and `#FF4500`).
- **Custom Interactive Cursor**: A custom-built cursor that trails the mouse and expands with a magnetic glow effect when hovering over interactive elements.
- **Micro-Animations & Transitions**: Smooth scroll reveal animations (`opacity` and `transform` translations) that bring elements to life as you scroll down the page.
- **Glassmorphism Elements**: Navigation bars and cards that utilize `backdrop-filter: blur()` for a sleek, modern, translucent look.
- **Noise Textures**: A subtle SVG noise overlay (`mix-blend-mode`) that adds depth and a tactile, editorial feel to the background.
- **Magnetic Buttons**: Call-to-action buttons that slightly attract towards the cursor on hover, creating a satisfying and playful interaction.
- **Dynamic Marquee**: An infinite scrolling skill ticker that adds continuous movement to the page without being distracting.

## 🛠️ Technologies Used

- **Frontend Structure**: HTML5
- **Styling**: Tailwind CSS (via CDN) & Custom Vanilla CSS (for complex animations and theme variables)
- **Interactivity**: Vanilla JavaScript (ES6+)
- **Typography**: Google Fonts (`Cormorant Garamond` for headings, `DM Sans` for body text)
- **Icons**: Iconify

## 🚀 Setup & Installation

Since this is a static website, no complex build process is required. 

1. **Clone the repository:**
   ```bash
   git clone https://github.com/rutuja-codes-ux/Personal_Portfilio.git
   ```

2. **Open the project:**
   Simply open the `index.html` file in any modern web browser (Chrome, Firefox, Safari, Edge).

   *Alternatively, if you have VS Code installed, you can use the **Live Server** extension for a better development experience.*
